== Description ==  
**Stop fake and spam reviews** on your WooCommerce store with advanced protection. This plugin automatically filters suspicious reviews using AI, IP blocking, and custom rules to maintain your store’s credibility.  

**Key Features:**  
✔ AI-powered spam detection  
✔ Block reviews from suspicious IPs/bots  
✔ Custom keyword blacklist  
✔ CAPTCHA integration (Google reCAPTCHA/hCAPTCHA)  
✔ Whitelist trusted customers  
✔ Export/import spam patterns  

== Installation ==  
1. Upload the plugin ZIP file via **WordPress Admin → Plugins → Add New → Upload**.  
2. **Activate** the plugin.  
3. Go to **WooCommerce → Settings → Anti-Spam Pro** to configure.  

== Configuration ==  
After activation:  
1. **Enable** spam filtering in plugin settings.  
2. Set up CAPTCHA (recommended).  
3. Add high-risk keywords (e.g., "buy cheap", "viagra").  
4. Adjust sensitivity (Low/Medium/High).  

== Frequently Asked Questions ==  
**Q: Does this work with WooCommerce’s native reviews?**  
A: Yes, it filters both native and third-party review systems.  

**Q: Can I whitelist loyal customers?**  
A: Yes, use the "Trusted Users" tab to add exempted emails/IPs.  

**Q: How does AI detection work?**  
A: The plugin analyzes review text patterns (e.g., generic praise, links) and user behavior.

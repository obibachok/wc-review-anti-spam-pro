## 🔍 Overview
A premium WordPress plugin that **automatically detects and blocks** spammy/fake product reviews using:
- **AI-powered pattern recognition**
- **IP/geolocation blocking**
- **Custom keyword blacklists**
- **CAPTCHA integration** (reCAPTCHA/hCAPTCHA)
- **Trusted customer whitelisting**

**→ Prevent review bombing**  
**→ Maintain product credibility**  
**→ Reduce manual moderation time**

---

## 🚀 Installation
1. **Download** the plugin ZIP from your purchase email.
2. In WordPress:  
   `Plugins → Add New → Upload Plugin` (select the ZIP file)
3. **Activate** the plugin.
4. Configure under:  
   `WooCommerce → Settings → Anti-Spam Pro`

---

## ⚙️ Configuration
| Setting Tab         | Key Features                              |
|---------------------|------------------------------------------|
| **General**         | Enable/disable spam filtering            |
| **CAPTCHA**         | Google reCAPTCHA or hCAPTCHA integration |
| **Keyword Rules**   | Add blocked phrases (e.g., "buy cheap")  |
| **IP Blocking**     | Ban suspicious IP ranges                 |
| **Whitelist**       | Exempt trusted customers                 |

## ❓ FAQ
### 🤖 How does AI detection work?
Analyzes review text for:
- Generic praise ("Great product!")
- Suspicious links
- Duplicate content patterns

### 🔒 Can I whitelist loyal customers?
Yes! Add approved emails/IPs in the **Whitelist** tab.

### 📊 Does it work with review plugins?
Compatible with:
- WooCommerce native reviews
- Yotpo
- Judge.me
- WP Customer Reviews
